🌾 SmartAgroSupplyChainAISmartAgroSupplyChainAI is a full-stack platform for managing agricultural product inventory and shipments, powered by Groq AI (LLaMA 3.3-70b) and AI Tool Calling. Users can issue commands in plain natural language—the AI automatically selects the correct tool and executes the operation against the database. The system is fully controllable via both a REST API and a visual Streamlit dashboard.🚀 Key Features🤖 AI Tool Calling: LLaMA 3.3-70b automatically selects and executes the right tool from 4 core functions.🌾 Inventory Management: Add, view, and deduct warehouse stock.🚚 Shipment Tracking: Create shipments and track status (In Transit → Delivered).⚡ Redis Caching: /inventory/ and /shipments/ endpoints cached for 1 hour with auto-invalidation.🧹 Data Cleaning: Regex pipeline strips URLs, HTML tags, emails, and special characters before DB writes.📊 Streamlit Dashboard: 4-page interactive dashboard (Dashboard, Inventory, Shipments, AI Assistant).🐳 Docker Compose: 4 services (backend, frontend, postgres, redis) orchestrated with healthchecks.🛠️ Tech StackLayerTechnologiesBackendPython 3.11, FastAPI, AsyncGroq, SQLAlchemy (Async), PydanticAIGroq Cloud API (LLaMA-3.3-70b-versatile)Data LayerPostgreSQL 15, Redis 7, Regex (re)FrontendStreamlit, Plotly, Pandas, httpxInfrastructureDocker, Docker Compose📂 Project StructurePlaintextSmartAgroSupplyChainAI/

🔌 API EndpointsPOST /inventory/: Add a new product or increase existing stock.GET /inventory/: Retrieve all stock (Redis cached).POST /shipment/: Create a new shipment and deduct from stock.GET /shipments/: Retrieve all shipments (Redis cached).PATCH /shipment/{id}/deliver: Mark a shipment as "Delivered".POST /ai-task/: Natural language processing → AI Tool Calling.📊 Streamlit Dashboard PagesDashboard: Real-time overview with KPI cards, stock bar charts, and shipment status.Inventory: Detailed stock levels and management form.Shipments: Shipment tracking, destination analysis, and delivery status updates.AI Assistant: Natural language interface to trigger database operations via LLM.⚠️ DisclaimerThis application is intended for demonstration and informational purposes only.
